- ✅ Green/Black theme (replaced original blue)

## Tech Stack

- React 19
- TypeScript
- React Router DOM
- Vite
- CSS3 (Custom green/black theme)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
